# [機能名] — `/add-feature` 実行コマンド一覧

本書は [[feature-name]-plan.md](./[feature-name]-plan.md) の実装を [N] つの独立した `/add-feature` コマンドに分割したものである。各ステップは単独でレビュー・マージ可能な粒度に揃え、依存関係が前→後へ一方向になるよう順序付けしている。

**重要**: 各 `/add-feature` コマンドのプロンプトには「参照ドキュメント: `docs/ideas/[feature-name]-plan.md`」が含まれており、実装時には常に同プランを参照しながら該当ステップ範囲のみを実装する。プラン全体を一度に実装しないこと。

**前提**: [前提条件があれば記載 — 既存基盤・先行 PR など]

## 実行順の全体像

```
Step 1: [ステップ 1 タイトル]
   ↓   ← Step 1 完了時点で「[何が動くか]」
Step 2: [ステップ 2 タイトル]
   ↓   ← Step 2 完了時点で「[何が動くか]」
Step 3: [ステップ 3 タイトル]
       ← Step 3 完了時点で「[全機能完成]」
```

**ポイント**:

- [各ステップの価値・段階リリースの利点を 2〜3 行]
- 各ステップ後に `npm run lint:fix && npm run format && npm run build && npm test` が PASS することをゲートとする
